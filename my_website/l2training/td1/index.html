Rapport de projet Informatique - Conception d'une application d'agrégation de données

Introduction

GAgrégateur est un collecteur de données de pays (géographiques et démographiques). Pour ce faire, le logiciel utilise des algorithmes de lecture et de parcours de fichiers basiques.

Le résultat escompté est l'affichage des données et la possibilité de rechercher et de trier les informations selon des critères autre que le pays.

Après un mois et demi de travail, cette version de l'application affichera uniquement les données relatives au pays choisi.

Son fonctionnement est simple, il suffit de taper le nom du pays pour afficher ses informations. La liste de tous les pays est disponibles si besoin. La sauvegarde et le chargement de dernière consultation sont prêtes à l'emploi (via fichier texte).

Une version console CAgrégateur est également présente pour l'exécution du programme via le terminal.
