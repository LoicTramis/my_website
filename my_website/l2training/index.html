Mon site

Ce site web regroupe tous les travaux dirigés du module Développement Web.

Au semestre 4, l'UE Développement Web est coefficient 6 tout comme l'UE Séries.

Ce module introduit le langage HTML (XHTML1.0 strict & HTML5), l'inséparable CSS (2 & 3), et quelques notions de programmation web (PHP).

Les anciens travaux dirigés

Les TD 1 & 2 sont réalisés en XHTML, contrairement au reste qui est en HTML 5. De ce fait ils ne sont que partiellement intégrés au site et ne sont accessibles que depuis la page d'accueil.

Cliquez sur les images pour accéder au TD.

Les nouveaux travaux dirigés

À partir du 5ème TD, tous les TD seront intégrés au site accessible depuis la barre de navigation situé ici.
